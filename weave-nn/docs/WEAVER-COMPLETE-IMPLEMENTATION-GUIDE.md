### SOP Execution Pattern

Every SOP follows the 7-phase execution:

```typescript
async function executeSOP(sopName: string, inputs: any) {
  // 1. Perception: Retrieve past executions
  const context = await learningLoop.perceive({ sop: sopName, inputs });

  // 2. Reasoning: Generate optimal plan
  const plan = await learningLoop.reason(context);

  // 3. Coordination: Initialize MCP coordination (optional)
  await claudeFlow.swarm_init({ topology: plan.topology });

  // 4. Execution: Spawn real agents via Claude Code Task tool
  const agents = await spawnAgents(plan.agents);
  const result = await executeAgents(agents, plan);

  // 5. Reflection: Analyze outcomes
  const lessons = await learningLoop.reflect(result);

  // 6. Memory: Store experience
  await learningLoop.memorize({ sop: sopName, result, lessons });

  // 7. Artifacts: Save outputs in vault
  await saveArtifacts(result);

  return result;
}
```
